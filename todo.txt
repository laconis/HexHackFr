⚙️ 3. Configurer le chroot dans SSH

Éditer :
Code

sudo nano /etc/ssh/sshd_config

Ajouter à la fin :
Code

Match Group sftpusers
    ChrootDirectory /sftp/%u
    ForceCommand internal-sftp
    X11Forwarding no
    AllowTcpForwarding no

Cela signifie :

    chaque utilisateur est enfermé dans /sftp/<nom>

    pas de shell, uniquement SFTP

    pas de forwarding
